洛阳钧科机电设备有限公司

洛阳钧科机电设备有限公司成立于2010年，是一家专业从事机电设备销售、安装调试、维修保养、技术改造的高新技术企业。公司坐落于洛阳市高新技术产业开发区，拥有现代化办公场所和标准化生产车间。

公司主营各类电机、变频器、PLC控制柜、配电柜、工业自动化设备等，产品广泛应用于冶金、矿山、建材、化工、电力等行业。多年来我们始终坚持"质量为本、客户至上"的经营理念，为客户提供优质的机电设备整体解决方案。
